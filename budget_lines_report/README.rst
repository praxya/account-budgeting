Budget lines report
===================

Add a view on reporting for crossovered budget lines

Installation
============

To install this module, just follow basic steps to install an odoo module.


Configuration
=============

Not needed.

Usage
=====

Go to Reporting -> Budget -> Budget details and use as other BI reports.
